Présentation du projet

Dans le cadre d'un cours de web, nous avons réalisé le site Eventum. Eventum est un site web permettant de créer des événements et de les partager avec ses amis. Nous avons réalisés ce site en utilisant le framework Angular pour le front-end ainsi que PHP et SQL pour le backend.

Caractéristiques principales :

Le site comporte :

  • Un système d'amis avec lesquels il est possible d'avoir des messages privés.
  • Des évenements proposés par des utilisateurs, qu'ils soient public ou privés.
  • Un système pour inviter ses amis à des évenements !
  • Une conversation de groupe où il est possible de discuter avec les autres utilisateurs inscrits à l'évenement.
  • Un formulaire pour ajouter un nouvel évenement.
  • Un "agenda" pour avoir accès aux évenements que l'on a proposé et ceux auquels on est inscrit.
  • Un profil où l'on peut modifier ses informations, accéder à sa liste d'amis, les demandes d'amis en attente et se déconnecter.
  • Une page d'accueil qui propose les événements que l'utilisateur est le plus susceptible d'aimer.
